# Interactive Computer Graphics
## Project C - Space Shooter

### 遊戲簡介

主角搭乘之太空船遭受隕石撞擊，受困於外太空，並且太空船防衛機制失控，因此主角需一面破壞、閃躲砲台、守衛...，一面前往控制室取回太空船主控權，以返回地球。

### 操作說明

`Tab`: 開啓技能表

`Space`: 跳躍

右鍵: 瞄準

左鍵: 射擊

`Q` `Esc`: 暫停選單

`W` `A` `S` `D`: 移動角色

### 遊戲機制

- 躲避砲台射擊
- 主動射擊破壞砲台
